## 📌 Como Executar o Projeto

1 - Clone o repositório:

```bash
git clone https://github.com/Nicolas121221/Projeto-Entra21.git
```

2 - Acesse a pasta do projeto:

```bash
cd frontend
```

3 - Instale as dependências.

```bash
npm install
```

4 - Inicíe o Svelte.

```bash
npm run dev
```
